Clés API

Accès programmatique en lecture seule aux données du desk (news, calendrier, force des devises, biais, régime de marché, taux, rapports). Chaque clé est limitée à 60 requêtes/minute. La clé complète n'est affichée qu'une seule fois à la création — elle est ensuite stockée hachée (irrécupérable).

Documentation — API v1

Base : https://desk.datatradingpro.com · Authentification : header X-API-Key: <clé> (ou Authorization: Bearer <clé>) · Réponse : { ok, data, meta } · Lecture seule, jamais de clé dans l'URL.

curl -H "X-API-Key: dtp_votrecle" https://desk.datatradingpro.com/api/v1/news?limit=5
EndpointDescriptionParamètres
GET /api/v1/statusÉtat de la clé (usage, quota) + liste des endpoints—

GET /api/v1/riskRégime de Marché (risk-on / risk-off, score)—
